Polis Admin Console
===================

Configuration
-------------

Install the NVM following the instructions: [NVM Installation Guide](https://github.com/creationix/nvm#install-script)

```
$ nvm install 6.2.0
$ npm install
```

### Common Problems

If you having troubles with npm dependencies try:

```
$ npm cache clear
$ npm install
```

Running
-------

```
$ nvm use 6.2.0
$ nvm start
```
